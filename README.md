# 💍 Wedding Venue Intelligence — Web Scraping & Data Analysis

## 📌 Project Summary
This project demonstrates an end-to-end data analysis pipeline by extracting real-time wedding venue data from online platforms and transforming it into actionable insights.

The objective is to analyze venue pricing, location trends, and customer preferences using web scraping and exploratory data analysis (EDA).

---

## 🎯 Business Objective
- Identify pricing trends across locations  
- Analyze venue popularity using ratings  
- Understand location-based demand patterns  
- Support decision-making for venue selection  

---

## 🛠 Tech Stack

**Programming & Analysis**
- Python  
- Pandas, NumPy  

**Web Scraping**
- Requests  
- BeautifulSoup  

**Visualization**
- Matplotlib  
- Seaborn  
- Plotly  

**Environment**
- Jupyter Notebook  

---

## 🌐 Data Acquisition
- Collected real-time data from wedding venue listing platform (WedMeGood)  
- Extracted structured data including:
  - Venue Name  
  - Location  
  - Price Range  
  - Ratings  

---

## 🧹 Data Preprocessing
- Cleaned inconsistent and missing values  
- Standardized price formats and removed symbols  
- Converted features into appropriate data types  
- Eliminated duplicates  
- Prepared analysis-ready dataset  

---

## 📊 Exploratory Data Analysis

### 🔹 Pricing Analysis
- Distribution of venue prices  
- Identification of premium vs affordable venues  

### 🔹 Location Insights
- Location-wise price comparison  
- High-demand areas identification  

### 🔹 Rating Analysis
- Distribution of ratings  
- Top-performing venues  

### 🔹 Relationship Analysis
- Correlation between price and ratings  
- Trend visualization using scatter plots  

---

## 📈 Key Findings
- Pricing varies significantly across different locations  
- Premium venues are concentrated in high-demand areas  
- Higher-rated venues generally fall in mid-to-high price range  
- Location plays a crucial role in pricing and demand
